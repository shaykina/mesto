# Проект: Место

Одностраничный сайт содержит 2 секции, шапку и подвал. В проекте использованы **флексбокс-вёрстка**, реализованы изменения прозрачности ссылок при наведении. Файловая структура CSS реализована по методологии **БЭМ** (Nested). С помощью **javascript** прописано открытие и закрытие попапа, а также редактирование информации профиля.
